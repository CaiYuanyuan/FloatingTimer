# FloatingTimer（悬浮计时器）

<br />

**使用说明**：因为找不到符合自己需求的悬浮计时器，所以自己动手弄了一个，有需要的小伙伴可以直接[下载压缩包文件](https://download.csdn.net/download/VickyTsai/92756110?spm=1001.2014.3001.5503)使用。
- **系统**：仅支持 Windows 系统。
- **便携版本**：不需要安装，解压缩后，双击 **FloatingTimer-1.0.0-portable.exe** 文件直接使用。
- **快捷方式**：把.exe 放其他盘，右击.exe 文件→ 选择“创建快捷方式”，把快捷方式放到桌面，双击快捷方式使用。
- **固定到任务栏**：双击.exe文件运行后，右击桌面最下方的任务栏图标，选择“固定到任务栏”，随时用随时开。
- **注**：部分电脑的安全卫士可能会“报错、木马”提醒，直接选择“信任、加白名单”即可，因为是免费使用的小应用，没有购买数字签名证书，容易被报错。源码都在这里，也可以自己检查代码、打包成桌面应用。

<br />

**适用场景**：想自律，但总容易分神，这个时候就需要“提醒”。
  - 可以给每天的 “**最高优先级的目标**” 设置一个时长，比如 4小时、6小时、8小时、10小时，计时器会一直悬浮在桌面上，提醒我们“当前的学习时长”，让我们时刻做到心里有数，专注于最高优先级的目标。
  - 当学习时长达到后，则可以去做其他不那么重要的琐事，优先保证每天的学习时长，预防拖延症，提高专注力。

## 2. 基础功能简介

⑴ **悬浮、可拖动**：计时器能悬浮在桌面，并可拖动到任意位置。

⑵ **正计时、倒计时**：能正计时、倒计时，都可以从指定时间开始，倒计时结束有简单的提醒。

⑶ **暂停**：可以暂停，暂停后点“开始”继续计时。

⑷ **重置**：时间可以重置为0 ，有确认提醒，防误触。

⑸ **关闭**：点 x 可关闭窗口。
